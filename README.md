# Tanadi AI

Tanadi AI is an AI-powered saving assistant designed to help users manage money, build saving habits, and achieve financial goals.

## Features
- AI saving recommendations
- Spending analysis
- Savings goal tracking
- Smart alerts and notifications

## Technology
- Python
- Machine Learning
- Mobile Integration

## Dataset
The system uses financial transaction data to generate AI saving recommendations.

- Mobile app
- Bank API integration
